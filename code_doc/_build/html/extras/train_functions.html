train_functions module

train_functions.train(env, agent, data, learn_plot, episode_mem_size, n_episodes, n_steps, model_path, save_training=True, vis_compare=12, mnist=False, speed=False)

train Trains a specific model in an environment

Parameters

  • env (environment.Environment) – The environment where the agent is moving

  • agent (nn_agent.Agent) – The agent moving in the environment

  • data (data.ai_data.Ai_Data) – The data for the environment

  • learn_plot (matplotlib.pyplot.Figure) – The matplotlib plot where to plot the learning curve

  • episode_mem_size (int) – How big the replay buffer is allowed to be

  • n_episodes (int) – How many episodes to train the model

  • n_steps (int) – Allowed number of steps the agent is allowed to take per episode

  • model_path (str) – The path to the file where the checkpoint files of the model should be saved

  • save_training (bool, optional) – Should this training be saved to a checkpoint file, defaults to True

  • vis_compare (int, optional) – How often the visual comparison plot should be shown to the developer, defaults to 12

  • mnist (bool, optional) – Is this a mnist variation, defaults to False

  • speed (bool, optional) – Is this a speed variation, defaults to False
